# Roadmap

This document outlines the planned direction for Sven Family. Priorities may shift based on community feedback and user needs.

## Q2 2026

- [x] Auto-detect production services with health-check fallback in Studio
- [x] Community frontend production deployment (Vercel)
- [x] Standardize frontend env configuration across all packages
- [ ] Improve cross-service observability and distributed tracing
- [ ] Standardize API contracts and integration tests across systems
- [ ] Enhance test coverage for core backend and frontend modules

## Q3 2026

- [ ] Improve developer onboarding and local environment tooling
- [ ] Add automated cross-system data consistency checks
- [ ] Performance optimization for data collection and stats services

## Future

- [ ] Plugin system for Studio workflows
- [ ] Public API for third-party integrations
- [ ] Multi-tenant support

## How to Contribute

Have an idea or feature request? Open a [GitHub Issue](https://github.com/laishiwen/sven-family/issues) or start a discussion. We'd love to hear from you.
