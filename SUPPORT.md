# Support

## Getting Help

If you need help with Sven Family, here are your options:

### Documentation

- [README](README.md) — project overview and quick start
- [Contributing Guide](CONTRIBUTING.md) — how to set up development and contribute

### Questions & Discussions

**[GitHub Issues](https://github.com/laishiwen/sven-family/issues)**

For usage questions, best practices, and general discussion.

### Bug Reports & Feature Requests

Open an issue on [GitHub Issues](https://github.com/laishiwen/sven-family/issues).

- Bug reports should include reproduction steps and environment details
- Feature requests should describe the problem and proposed solution

### Security Issues

See [SECURITY.md](SECURITY.md) for reporting vulnerabilities. Do not use public channels for security issues.

### Community

Follow the [Code of Conduct](CODE_OF_CONDUCT.md) in all interactions.
